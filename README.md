# i4.0-sistemas-ciberfisicos-e-gemeos-digitais-industriais
Projeto focado em fundamentos da Indústria 4.0, com ênfase em Gêmeos Digitais via modelo AAS (Asset Administration Shell). Inclui práticas com sensores, OPCUA, MQTT, Python assíncrono e bancos NoSQL para criação de modelos digitais interoperáveis..
